1. The Square Root Trap (Explained Simply)
Imagine a right triangle. To find the length of the longest side (the distance between two points), we use the Pythagorean theorem you learned in school: A² + B² = C²
•
C is the actual distance.
•
To find C, the computer has to do this math: C = √(A² + B²)
The Problem: Calculating a Square Root (√) is incredibly difficult for a computer. It takes the CPU a long time to guess and calculate it. Doing this hundreds of times a second causes your phone to overheat.
The "Cheat Code" (sqrMagnitude): Let's say we want to check if the enemy is closer than 5 meters. Instead of forcing the computer to calculate the square root to find C, we use basic algebra.
We ask the computer: "Hey, is C² less than 5²?" 5 squared (5 * 5) is 25.
So, instead of checking if Distance < 5, we check if SquaredDistance < 25. The math result is exactly the same, but the computer skips the heavy square root calculation entirely! In Unity, Vector3.sqrMagnitude is how you get that C² (Squared Distance) value.
